# Example: unified_generator_demo

**Crate**: `cfd-schematics`  **Run** (from crates/cfd-schematics): `cargo run --example topology/unified_generator_demo`

Demonstrates the unified topology generator: bifurcations, serpentines, Venturi, and split-tree topologies all produced from a single parameterised API call.
