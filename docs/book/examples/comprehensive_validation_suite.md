# Example: comprehensive_validation_suite

**Crate**: `cfd-validation`  **Run**: `cargo run -p cfd-validation --example comprehensive_validation_suite`

Runs the complete CFDrs validation matrix:  cavity flow (Ghia Re=100/400/1000), pipe Poiseuille, Kovasznay flow, and lid-driven cavity convergence.  All cases must pass before a release.
