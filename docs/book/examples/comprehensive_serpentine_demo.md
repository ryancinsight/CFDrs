# Example: comprehensive_serpentine_demo

**Crate**: `cfd-schematics`  **Run** (from crates/cfd-schematics): `cargo run --example comprehensive_serpentine_demo`

Generates the full serpentine-channel schematic family across aspect ratios, turn counts, and channel widths.  Writes SVG output to the working directory.
