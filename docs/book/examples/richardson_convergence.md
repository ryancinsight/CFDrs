# Example: richardson_convergence

**Crate**: `cfd-validation`  **Run**: `cargo run -p cfd-validation --example richardson_convergence`

Verifies second-order spatial convergence of the pressure solver via Richardson extrapolation on four successively refined grids (h, h/2, h/4, h/8).
