# Example: spectral_poisson_3d (cfd-3d)

**Crate**: `cfd-3d`  **Run**: `cargo run -p cfd-3d --example spectral_poisson_3d`

Solves the 3-D Poisson equation spectrally on a periodic cube, validating the FFT-based pressure solver against an analytical manufactured solution.
