# Milestone 12 Topology SSOT Audit

## Purpose

This note records the architectural contract for Milestone 12 selective-routing work after the SSOT refactor. It is written to complement the generated JSON and markdown ledgers emitted under `report/milestone12/`.

## Topology Authority

`cfd-schematics` is the sole authority for selective-routing topology, geometry authoring, and venturi promotion.

- `cfd-optim` may construct Milestone 12 request objects and evaluate candidate physics.
- `cfd-optim` must not manually construct selective `NetworkBlueprint` graphs or bootstrap venturi geometry in place.
- Option 1 to Option 2 promotion is canonical only when it passes through `cfd_schematics::promote_milestone12_option1_to_option2`.
- `cfd-mesh` accepts selective blueprints only when they carry canonical topology metadata, render hints, and `create_geometry` provenance.

## Mesh Interpretation

For selective split trees, `cfd-mesh` now treats the authored blueprint polyline as the geometric contract. The mesh pipeline no longer synthesizes a second selective layout from graph depth ordering. This matters physically because treatment-lane branch curvature, asymmetric daughter-width transitions, and venturi placement must remain identical between schematic analysis and downstream CFD meshing.

## Optimization Interpretation

Milestone 12 still has three scientific tracks:

1. Option 1: asymmetric split routing for residence time and differential RBC/WBC/CTC partitioning.
2. Option 2: venturi-enabled selective cavitation on the treatment lane.
3. GA refinement: in-place curvature and throat refinement of venturi-capable seeds.

The GA score is now explicitly coupled to both routing and cavitation physics. It rewards:

- treatment-lane enrichment of circulating tumor cells,
- treatment-zone residence time,
- selective cavitation support with reduced healthy-cell exposure,
- Dean-vortex intensity from curvature-driven secondary flow at bend apices,
- lineage-preserving in-place refinement.

The relevant curved-channel mechanism is the Dean secondary-flow instability generated by centrifugal forcing in a curved duct. In this context, bend apices and outer-wall regions are the most relevant sites for curvature-assisted focusing prior to or near a venturi throat.

## Evaluation And Audit Artifacts

Successful evaluations must never collapse validity and ranking into `score == 0`.

- Successful evaluations carry a strictly positive ranking score.
- Screening state is explicit in the evaluation payload.
- Invalid candidates remain hard errors.
- Generated ledgers are split into `eligible`, `screened_out`, and `errored` JSON outputs.

The canonical helper is `cfd_optim::write_goal_audit_report`, which writes:

- `<stem>_eligible.json`
- `<stem>_screened_out.json`
- `<stem>_errored.json`
- `<stem>_summary.md`

## Verification Expectations

- Use `cargo nextest run` with the workspace timeout profile in `.config/nextest.toml`.
- Validate Milestone 12 request materialization, canonical Option 1 promotion, GA seed-pair construction, and selective mesh provenance rejection.
- Treat any zero-score path in successful evaluations as a defect requiring investigation rather than as an acceptable ranking result.
